# Support and Community

If you need help or want to get involved, here are the main channels:

- GitHub Issues: https://github.com/advatar/browser/issues
  - Report bugs and request features. Please include logs and reproduction steps.
- GitHub Discussions: https://github.com/advatar/browser/discussions
  - Ask questions, propose ideas, and share tips.
- Security: See the Security section in `README.md` or email security@yourdomain.com.

Before opening an issue, please check the [Troubleshooting Guide](docs/TROUBLESHOOTING.md) and existing discussions.
